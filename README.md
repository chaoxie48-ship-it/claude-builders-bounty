# Changelog Generator

Generate a structured CHANGELOG.md from git history.

## Setup (3 steps)

1. Save this script as `changelog.sh`
2. Make it executable: `changelog +x changelog.sh`
3. Run: `./changelog.sh`

## Usage

```bash
./changelog.sh
```

Output will be saved to `CHANGELOG.md`
